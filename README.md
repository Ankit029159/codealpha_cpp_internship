# codealpha_cpp_internship

## To-Do List Application
A simple console-based to-do list application written in C++ that allows users to manage tasks efficiently.

### Key Features

- Add Tasks: Users can add new tasks with a description to their to-do list.

- Mark Tasks as Completed: Tasks can be marked as completed by specifying their number in the list.

- View Tasks: Users can view all tasks along with their statuses (Pending or Done).

- Console-based Interface: Simple and intuitive text-based menu for interaction.

### Technologies Used

- C++ Standard Library: Utilizes iostream, vector, and string for input/output operations, dynamic task management, and handling text.
